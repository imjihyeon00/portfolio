GAME

jQuery로 제작한 뒤집어진 카드들 중 동일한 한 쌍의 카드를 찾아내는 게임입니다. 게임을 시작할 때 랜덤으로 셔플된 카드들의 위치를 한번 보여 준 뒤, 5번의 기회 동안 동일한 카드들을 찾아내면 성공 입니다. 5번의 기회를 모두 사용하면 시도가 1씩 증가하고 시도 횟수에 따라 성공했을 때의 문구가 다르게 나타나는 것을 확인 할 수 있습니다. 사이트의 우측 상단에 있는 아이콘을 누르면 음악이 재생되고 카드를 뒤집어 맞추거나 틀렸을 때 효과음이 들립니다.

VIEW

SEARCH

jQuery로 작성한 CSS 속성을 입력하는 search 게임 입니다. 시작하기 버튼을 클릭하면 setInterval을 이용해 1분간 카운트를 시작되며 input 박스에 텍스트를 입력 할 수 있게 됩니다. 텍스트를 입력하면 input이벤트가 발생되며 CSS 속성 배열의 값들과 비교하여 정답이면 객체의 값을 변경 후 화면 아랫쪽에 텍스트가 추가되고, 게임이 종료되면 맞추지 못한 CSS속성이 밑에 나타납니다.
